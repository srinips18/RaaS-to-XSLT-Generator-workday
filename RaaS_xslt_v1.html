Summarise & Roll Up ● Active
Click to configure grouping
Assign each column a role to summarise and roll up repeating rows into one output row per person / group. Identity columns stay the same across all rows (Employee ID, Name) — one output row per unique combination. Total / Calculate columns are computed across all rows in each group (add up amounts, count rows, find highest value, etc.). Columns with no role are left out of the rolled-up output.

Output Format
⚠️ Workday EIB cannot generate native Excel files via XSLT.
The correct approach is to generate a CSV that Excel opens natively. Configure EIB as follows:
Get Data → Alternate Output Format: CSV
Deliver → File Type: CSV
Deliver → File Name: YourReport.csv
✅ XSLT outputs comma-delimited CSV — Excel opens it directly
